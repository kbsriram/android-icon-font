Information about the font

Please keep in mind that this font is not hinted, and so works best on higher density displays. You can download AndroidIcons.ttf and use it in your Android application or as a webfont (as this page does). There is a Github repository so you can also tweak the source directly if you prefer.

This font was created by vectorizing icons from the Android 4.0 Core Icon Set, and is made available under the SIL Open Font License.
